# Script Formats

## Short-Form Explainer, 30-60s

Structure:

1. Pattern interrupt hook.
2. Pain or misconception.
3. New mechanism.
4. Concrete example.
5. CTA or next step.

Best for: TikTok, YouTube Shorts, Instagram Reels, X video, LinkedIn video.

## Portfolio Demo, 30-90s

Structure:

1. Business problem.
2. Workflow transformation.
3. Before/after output.
4. Stack or method.
5. Offer.

Best for: service portfolio, pinned X/LinkedIn, landing page hero video.

## Tutorial, 3-8m

Structure:

1. Outcome preview.
2. Prerequisites.
3. Step-by-step build.
4. Common failure point.
5. Recap and next action.

Best for: YouTube, course lead magnet, docs.

## Case Study, 2-5m

Structure:

1. Situation.
2. Constraint.
3. Intervention.
4. Result.
5. Reusable lesson.

Use anonymized examples when client or personal data may be sensitive.
